SmartCart App

SmartCart is a feature-rich Android grocery shopping application built using Java and Firebase. It offers seamless user registration and login with Firebase Authentication, and real-time product listing and cart management using Firebase Realtime Database. Key features include:

  • User sign-up, login, and password recovery using Firebase Authentication.
  • Adding products to the shopping cart with quantity management.
  • Cart screen displays total price and itemized list of selected items.
  • Support for automatic detection of user house number during login.
  • Splash screen with animated logo and tagline.
  • Modern and responsive UI following material design principles.
This app demonstrates full integration of front-end and back-end in a mobile environment.

Hangman Game

A fully interactive Hangman game developed in C++ using the SFML (Simple and Fast Multimedia Library). The game offers a playful user interface and supports both mouse and keyboard inputs. Features include:

  • Word guessing game with a randomly selected word from a predefined list.
  • Difficulty selection screen controlled via keyboard input.
  • Main game screen uses mouse input to click on alphabet letters.
  • Visual feedback with dashes (e.g., - - - - -), red for wrong guesses and green for correct ones.
  • Hint feature reveals one letter after a wrong attempt (optional).
  • Sound effects for correct/incorrect guesses using SFML audio module.
  • Victory or game over screen appears after each round with a restart option.
A good demonstration of object-oriented design and real-time event handling in game development.
